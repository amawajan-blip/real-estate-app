# Changelog

## [1.0.0] - 2024-01-01

### Added
- Initial release of مدير عقاراتك | Real Estate Manager
- Full property management (add, edit, delete properties and units)
- Tenant management with primary and sub-tenant support
- Payment tracking (rent, electricity, water, tax)
- Contract upload and file management stored locally
- Dashboard with financial overview and recent activity
- Reports with income/expense charts and occupancy rates
- Database backup and restore (export/import)
- Arabic (RTL) and English language support
- Light and dark theme
- Full offline operation — no internet required
- Native Windows installer and portable executable
